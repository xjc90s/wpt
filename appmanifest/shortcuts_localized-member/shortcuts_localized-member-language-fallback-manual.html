Testing language-only fallback for shortcut name_localized and icons_localized

Set your browser locale to one that includes a region subtag not present in the manifest (e.g. en-US, en-GB, fr-CA, or ja-JP), then install this page as a web application. To pass, the browser must fall back to the language-only key and the shortcut must be named "pass" with a green check mark icon (pass.png).
